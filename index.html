¿Qué es Dream City?

Dream City es una propuesta de ciudad inteligente diseñada para ser completamente autosustentable. Nuestro enfoque está en el uso de energías limpias, especialmente energía solar, para alimentar todas las infraestructuras urbanas, promoviendo la sostenibilidad y la innovación tecnológica.

El objetivo de este proyecto es desarrollar una maqueta funcional de una ciudad inteligente a escala, incorporando tecnologías de automatización y control mediante plataformas como Arduino. Esta maqueta servirá como herramienta educativa para demostrar cómo las soluciones tecnológicas pueden integrarse en entornos urbanos para mejorar la eficiencia y calidad de vida.
